# Git Conventions

- Commit format: `<type>: <description>` (one-liner)
- Types: feat, chore, doc, fix, infra
- Scan repo's commit history first to match existing style

# Git pull-request initial status
- Regardless of Git provider platform, always create pull request in draft status initially unless
  - There's a specific command in prompt telling the otherwise.
  - Git provider platform doesn't support draft status.
